## ‚ö° Installation & Run

1. Clone the repository

```bash
git clone https://github.com/khushi100092/weather-app.git
cd weather-app
```

2. Install dependencies

```bash
npm install
```

3. Create `.env` file in root folder

```env
API_KEY=YOUR_OPENWEATHERMAP_API_KEY
PORT=3000
```

4. Run the app

```bash
node index.js
```

5. Open in browser:

```
http://localhost:3000
```
